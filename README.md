# Recap of My Backend Development Class
This repository contains a recap of my backend development learning journey so far. It documents what I have covered each week and the skills I have developed.

## Week 1 – Foundations
In Week 1, we learned how the web works from the moment a user enters a URL into a browser to how the server responds with data. I understood the roles of DNS, firewalls, load balancers, web servers, application servers, and databases.
We installed Ubuntu using WSL and learned what Linux is, its history, and why it is open-source and widely used.

We practiced Linux commands such as:
- Creating and removing directories
- Creating, renaming, and deleting files
- Navigating between directories
- Checking the current directory location

We also started Git and GitHub, where we learned how to:
- Configure username and email
- Add files
- Commit changes
- Push to GitHub

## Week 2 – Deeper into Linux
In Week 2, we explored the Ubuntu terminal more deeply and understood how the shell works.
We learned about file permissions and how to control who can read, write, or execute files.
We also worked with package management using `sudo` and installed different packages such as:
- figlet
- cowsay
- and other terminal tools

## Week 3 – Introduction to Python
In Week 3, we started learning Python programming. We talked about variables, data types, typecasting.

## What i've learned
- Understanding web architecture and client-server communication
- Using the Linux command line confidently
- Managing files and directories in Linux
- Understanding file permissions and package management
- Using Git for version control
- Creating branches and pull requests
- Resolving merge conflicts
- Organizing project structure properly
- Writing Markdown documentation
- Beginning Python programming.
