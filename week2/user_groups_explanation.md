Field Explanation

dev_alice:x:1001:1001::/home/dev_alice:/bin/bash
dev_bob:x:1001:1001::/home/dev_bob:/bin/bash

1. dev_alice, dev_bob - Username : The login name of the user

2. x - password : Password of the user that is encrypted

3. 1001, 1002 - User id (UID) : A unique numeric identifier assigned to the user

4. 1001, 1002 - Group ID (GID) : The numeric ID of the user's primary group

5. :: - Comment : An optional field that can store additional user information

6. home/dev_alice, home/dev_bob - Home directory : The default directory the user is placed in after logging in

7. /bin/bash - Shell : The default shell that runs when the user logs in
